VR pics viewer For HTC Vive/Oculus Rift


list pics directories in filepaths.txt
> c:\images
> c:\pics
etc.

supports .jpg, .png, .wav (audio)

controls
-----------------

menu - next directory

trackpad - toggle fap/edit mode

trigger - move/scale

grip - delete pic from scene

trigger+trackpad l/r - change sound
